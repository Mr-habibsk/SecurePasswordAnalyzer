# 🔐 Secure Password Analyzer

A **Cyber Security Project** built using **C# and WPF** that analyzes password strength and estimates how long it would take to crack the password using brute-force attacks.

This tool is designed with a **Hacker-style terminal UI and Matrix animation background** to simulate a real cyber security environment.

---

## 🚀 Features

* 🔑 Password Strength Analyzer (Weak / Medium / Strong)
* ⏱ Estimated Password Crack Time
* 🖥 Hacker Terminal Style Interface
* 🟢 Matrix Rain Animation Background
* ⚡ Brute Force Attack Simulation
* 📊 Security Scan Output in Terminal
* 🎨 Custom Modern UI (Borderless Window)

---

## 🧠 How It Works

The software analyzes the password based on:

* Password length
* Uppercase characters
* Lowercase characters
* Numbers
* Special characters

Then it estimates how long a **brute force attack** might take to crack the password.

Example output:

```
> ENTER PASSWORD: Habib@123

> Initializing security scanner...
> Running brute force attack...
> Password length: 9
> SECURITY LEVEL: MEDIUM
> Estimated crack time: 4 hours
> Scan complete.
```
